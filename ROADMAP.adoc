:toc:
:toc-placement: preamble
:icons: font

This document outlines the development roadmap for rescript-poly-core, the foundation library for the Hyperpolymath ReScript ecosystem.

== Vision

PolyCore aims to be the definitive foundation library for building ReScript applications and MCP servers in the Hyperpolymath ecosystem. It provides battle-tested utilities, consistent patterns, and infrastructure that enables rapid development of reliable software.

== Current Status

**Phase: Alpha (v0.1.x)**

Core functionality is implemented and working. The library is suitable for experimental use but APIs may change before v1.0.

== Non-Goals

These are explicitly out of scope for PolyCore:

* **Web framework** - See poly-web for HTTP servers, routing, middleware
* **CLI framework** - See poly-cli for argument parsing, commands, TUI
* **Database ORM** - Each project should choose appropriate data layer
* **Node.js support** - Deno-only by design
* **React/UI components** - Frontend concerns are separate
